# DND Session Tracker

A fantasy-themed web dashboard for tracking and visualizing player statistics during a Dungeons & Dragons session.

This project provides a home page, a session dashboard, player stat cards, editable player data, SVG-based visualizations, and an end-session confirmation modal. It is built with vanilla HTML, CSS, and JavaScript.

## Overview

DND Session Tracker is a front-end web application designed for recording player performance during a DND session. The interface uses a dark fantasy visual style with glowing effects, animated particles, glassmorphism panels, and interactive statistic cards.

Users can select different players, update their session data, and view visual summaries for plot participation, hit rate, and total damage.

## Features

- Fantasy-themed landing page
- Session dashboard page
- Player list and player detail panel
- Real-time player stat editing
- Plot participation tracking
- Hit and miss tracking
- Automatic hit-rate calculation
- Total damage tracking
- Data reset controls
- Multiple visualization tabs:
  - Overview
  - Plot participation
  - Hit rate
  - Total damage
- SVG-based charts
- End-session confirmation modal
- Animated particle background
- Responsive layout

## Tech Stack

- HTML5
- CSS
- JavaScript
- SVG
- DOM Manipulation
